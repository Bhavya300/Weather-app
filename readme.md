# Weather App

A simple and interactive Weather Application that allows users to search for a city and view real-time weather information such as temperature and weather conditions.

## Features

- Search weather by city name
- Displays temperature in Celsius
- Shows weather description
- Error handling for invalid city names
- Clean and responsive user interface
- Real-time weather data using API

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
